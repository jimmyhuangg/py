## str connect
can use "+" or "*"
```
>>> 'Hello' + ' ' + 'world'
'Hello world'
>>> 'Hello' * 4
'HelloHelloHelloHello'

```
"list" and "tuple" can use it too!!

## 'in' and 'not in'
```
>>> a=4
>>> b=5
>>> li = [1,2,5]
>>> a in li 
False
>>> b in li
True
>>> a not in li
True
```

## '*'

```
>>> x = [1,2,3,4,5]
>>> a,*b,c = x
>>> a,b,c
(0,[1,2,3],4)
```
